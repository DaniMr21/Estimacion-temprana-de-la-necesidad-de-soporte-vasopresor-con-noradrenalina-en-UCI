# Estimación temprana de la necesidad de soporte vasopresor con noradrenalina en UCI 

Estudio de ML para estimar el riesgo de inicio de noradrenalina en UCI en las primeras horas de ingreso, usando constantes vitales y parámetros analíticos. Incluye dashboard interpretable que muestra el riesgo estimado y los factores clave por paciente.
