# Estimacion-temprana-del-riesgo-de-inicio-de-Noradrenalina-en-UCI
Modelo de ML para estimar el riesgo de inicio de noradrenalina en UCI en las primeras horas de ingreso, usando constantes vitales y parámetros analíticos. Incluye dashboard interpretable que muestra el riesgo estimado y los factores clave por paciente.
